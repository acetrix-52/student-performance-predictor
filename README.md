# Student Performance Predictor

## Problem
Students often cannot estimate their academic performance based on daily habits.

## Solution
This project uses Machine Learning to predict student marks based on:
- Study Hours
- Sleep Hours
- Attendance

## Features
- Predict marks using ML model
- Simple input system
- Optional web interface using Streamlit

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Streamlit

## How to Run

### Step 1:
Install libraries
pip install pandas scikit-learn streamlit

### Step 2:
Run basic model
python model.py

### Step 3 (Optional UI):
streamlit run app.py
